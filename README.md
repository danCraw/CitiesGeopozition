## FastAPI CRUD app with async DB and SQLAlchemy

## Run app
```bash
docker-compose -f docker-compose.dev.yml up
```
## Run tests
```bash
docker-compose -f docker-compose.test.yml run web pytest -vv
```
### Technologies used
FastAP
Pydantic
Swagger
SQLAlchemy
Typing
PostgreSQL
Pytest
